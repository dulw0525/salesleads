## 🎯 核心特性

### 双轨积分体系
- **充值积分**（不可提现）：用于申领线索
- **激励积分**（支持提现）：分享线索获得，需冻结30天

### 自动化风控
- 系统Hash查重，防止重复分享
- 第三方API实时校验（空号/停机/实名）
- 信誉分体系，阶梯式信用惩罚

### 合规保障
- 强制法律承诺函勾选
- 充值与提现严格解耦
- 提现资金定性为"劳务推广报酬"
